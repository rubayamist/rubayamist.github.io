Python Basics for Civil Engineering

A hands-on introduction to Python programming, built around real civil engineering calculations — from unit conversions to beam stress analysis.

1. Python Syntax & Basics 2. Variables, Data Types & Operators 3. Loops and Conditionals 4. Functions and Modules 5. Working with Engineering Data (NumPy Basics) 6. Simple Calculations: Load, Stress & Beam Analysis
Syllabus

Course Content

Module 1

Python Syntax & Basics

Python is a beginner-friendly programming language widely used in engineering for calculations, data analysis, and automation. This module covers variables, printing output, and comments.

Example: Print a simple greeting and store a project name in a variable.
# This is a comment
project_name = "Padma Bridge Rehabilitation"
print("Welcome to the project:", project_name)
Module 2

Variables, Data Types & Operators

Variables store data such as numbers (int, float), text (str), and true/false values (bool). Operators (+, -, *, /) let you perform calculations, such as unit conversions common in civil engineering.

Example: Convert a length of 25 meters into feet (1 m = 3.28084 ft).
length_m = 25
length_ft = length_m * 3.28084

print(f"{length_m} meters = {length_ft:.2f} feet")
Module 3

Loops and Conditionals

Loops repeat actions over a list of values, and conditionals (if/else) make decisions. Together they let you process many data points and flag values that need attention.

Example: Given a list of measured beam loads, print which ones exceed the allowable load of 50 kN.
beam_loads = [45, 52, 38, 60, 49]   # kN
allowable_load = 50                 # kN

for i, load in enumerate(beam_loads, start=1):
    if load > allowable_load:
        print(f"Beam {i}: {load} kN -> EXCEEDS allowable load!")
    else:
        print(f"Beam {i}: {load} kN -> OK")
Module 4

Functions and Modules

Functions let you package a calculation so it can be reused. Python's built-in 'math' module provides functions like square root, trigonometry, and constants such as pi.

Example: Write a function that calculates the cross-sectional area of a circular column.
import math

def circular_area(diameter):
    radius = diameter / 2
    return math.pi * radius ** 2

# Column with 0.4 m diameter
area = circular_area(0.4)
print(f"Cross-sectional area = {area:.4f} m^2")
Module 5

Working with Engineering Data (NumPy Basics)

NumPy is a library for working efficiently with numerical data, such as results from soil tests or material trials. It provides quick ways to compute averages, maximums, and standard deviations.

Example: Analyze a set of soil compressive strength test results (in kPa).
import numpy as np

test_results = np.array([210, 195, 230, 205, 220])  # kPa

print(f"Mean strength   = {np.mean(test_results):.2f} kPa")
print(f"Max strength    = {np.max(test_results)} kPa")
print(f"Min strength    = {np.min(test_results)} kPa")
print(f"Std. deviation  = {np.std(test_results):.2f} kPa")
Module 6

Simple Calculations: Load, Stress & Beam Analysis

This final module combines earlier concepts into a small program that calculates bending stress in a rectangular beam, using the standard formula: stress = (M * c) / I, where M is the bending moment, c is the distance to the extreme fiber, and I is the moment of inertia.

Example: A rectangular beam (width 0.2 m, depth 0.4 m) carries a bending moment of 60 kN·m. Find the maximum bending stress.
def rectangular_moment_of_inertia(width, depth):
    return (width * depth ** 3) / 12

def bending_stress(moment_kNm, width, depth):
    I = rectangular_moment_of_inertia(width, depth)   # m^4
    c = depth / 2                                       # m
    M = moment_kNm * 1000                                # N.m
    stress_Pa = (M * c) / I
    return stress_Pa / 1e6   # convert to MPa

width = 0.2   # m
depth = 0.4   # m
moment = 60   # kN.m

stress = bending_stress(moment, width, depth)
print(f"Maximum bending stress = {stress:.2f} MPa")
